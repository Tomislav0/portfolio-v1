Javi mi se

Unazad 5 godina bavim se fotografijom, a strast prema fotografiji prisutna je od malena. Specijaliziram se za fotografiranje portreta, događaja, noćnog života te poslovnu fotografiju, s ciljem da korisnicima pružim rezultate koji nadmašuju njihova očekivanja. Osim fotografiranja, moj interes leži u unaprjeđenju poslovanja klijenata kroz razvoj web rješenja i stranica, uključujući dizajn i pisanje programskog koda.

Koraci do unaprjeđenja vašeg poslovanja - web rješenje/stranica

01

Prvi korak - Kratkisastanak

Besplatni kratki sastanak gdje utvrđujemo što je vama potrebno i kako vam ja mogu pomoći.

02

Drugi korak - Definiranjezahtjeva

Kada ustanovimo da imam uslugu za Vas, zajedno prolazimo i definiramo što bi sve web rješenje/stranica trebala sadržavati. Diskutiramo o Vašim klijentima, postojećim dizajnovima, mogućim unaprjeđenjima...

03

Treći korak - Kreiranje prototipa

Kreiram prototip koji predstavlja “skicu” iz koje se može vidjeti dizajn web rješenja/stranice.

04

Četvrti korak - Razvoj web rješenja

Razvoj web rješenja/stranice slijedi nakon što utvrdimo da dizajn i sadržaj odgovara potrebama klijenta.

05

Peti korak - Isporuka i održavanje

Kad se utvrdi da je web rješenje/stranca razvijeno u skladu s Vašim potrebama i zahtjevima odrađuje deploy na web server. Ovisno o projektu web rješenje/stranca se održava prema dogovoru.
